# Imprint

**Imprint** is a personal life diary on a map with light social features. The product is diary-first: your map, memories (pins), chapters, and timeline are the core — social sharing, follows, and reactions are an optional layer on top, not the foundation. v1 is web-first (desktop, responsive mobile browser); native mobile comes later.

## Quickstart

```bash
corepack enable
pnpm install
pnpm dev          # starts apps/web
pnpm typecheck
pnpm lint
```
